# ❤️ Heart Disease Prediction System (Frontend)

## 📌 Overview

This is the frontend of an IoT-based Heart Disease Prediction System. It provides a modern and interactive user interface for monitoring health data, managing user profiles, and visualizing prediction results.

---

## 🚀 Features

* 🔐 User Authentication (Login & Signup)
* 👤 Profile Management (Static Health Data)
* 📊 Real-time Dashboard (Dynamic Sensor Data)
* 🧠 Prediction Panel (Risk Analysis)
* 🎨 Glassmorphism UI with Dark Theme
* 🔔 Custom Alert System

---

## 🛠️ Tech Stack

* React.js (Vite)
* Material UI
* JavaScript (ES6+)
* CSS (Glass UI Design)

---

## 📂 Folder Structure

```
src/
│
├── pages/        # Login, Signup, Profile, Dashboard
├── components/   # Reusable UI components
├── services/     # API calls
├── routes/       # App routing
├── styles/       # Theme & styling
```

---

## ⚙️ Installation

```bash
npm install
npm run dev
```

---

## 🌐 API Integration

The frontend communicates with a FastAPI backend for:

* Authentication
* Profile data
* Sensor data
* Prediction results
